Life Lab

A precise, unbounded Conway's Game of Life instrument. The grid is genuinely infinite — patterns are stored as live cells, so gliders and guns run off into empty space without hitting a wall. Edit cell-by-cell, stamp known patterns, or import any RLE.

Notes for exploration

Coordinates run to roughly ±33 million cells in each direction — far beyond any pattern you'll run by hand. RLE export produces standard B3/S23 files compatible with Golly and LifeWiki, so anything you discover here can be saved, shared, or opened elsewhere. Change the rule string to explore other life-like cellular automata on the same engine.
